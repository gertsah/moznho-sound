# Moznho Sound

Промо-лендинг для лейбла Moznho Sound с акцентом на:

- sticky-секции
- layered scroll-анимации
- выразительную типографику
- блок артистов и релизов на основе публично подтверждаемых источников

## Локальный запуск

Можно открыть `index.html` напрямую в браузере или поднять простой сервер:

```powershell
py -m http.server 3000
```

После этого страница будет доступна по адресу:

```text
http://127.0.0.1:3000
```

## Структура

- `index.html` — разметка лендинга
- `styles.css` — вся визуальная часть и адаптив
- `script.js` — scroll-driven анимации и reveal-эффекты
